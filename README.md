# An implementation of some classical machine learning algorithms

This is Python library implemented some Machine Learning algorithms using `numpy` in Machine Learning Course at HCMUT. 

## List of algorithms:
 - Linear Regression 
 - Logistic Regression 
 - Softmax Regression 
 - SVM 
 - PCA
 - LDA 
 - ID3 Decision Tree
 - HMM
